## Welcome

Learn the fundamentals of storing and manipulating data in a transactional database, essential for building applications with a foundational framework knowledge. Upon completion, you'll have the skills to read data, understand the benefits and limitations of transactional databases, manipulate database structures and data, and distinguish between various database systems.
